Buoi 1: Nen tang React cho Nhi

Buoi nay khong hoc nhieu thu lung tung. Chi tap trung 1 muc tieu: tao duoc ProductCard va render dung 10 item mock data bang map.

Muc tieu dat

  • Hieu component hoa UI.
  • Viet duoc JSX co du lieu dong.
  • Nhan du lieu qua props.
  • Gan event click dung cach.
  • Render list voi map + key.

Quy tac buoi hoc

  • Hoc xong 1 y moi moi chuyen sang y tiep theo.
  • Moi phan co mini demo 2 phut.
  • Neu bi ket > 3 phut thi hoi ngay mentor.
  • Cuoi buoi phai tu demo 3 phut.

Phan 1 - Tu duy component

Muc tieu: Nhi hieu vi sao React khong viet giao dien mot cuc lon.

HTML thuong

  • 1 file dai, kho tai su dung.
  • 100 san pham = lap code 100 lan.
  • Sua 1 cho phai sua khap noi.

React component

  • Tach nho: Header, ProductList, ProductCard.
  • Tai su dung ProductCard nhieu noi.
  • Code gon, de test, de bao tri.
Cau hoi goi mo: Neu can doi style ProductCard, sua 1 lan hay 100 lan?

Phan 2 - JSX can ban

JSX la cu phap gan giong HTML trong JavaScript. Du lieu dong dat trong dau ngoac nhon.

function ProductCard() {
  const name = "iPhone 15";
  return (
    <div className="card">
      <h3>{name}</h3>
      <p>Gia: 21.490.000d</p>
    </div>
  );
}
  • Dung className thay vi class.
  • Tra ve 1 khoi goc duy nhat.
  • Khong viet if/for truc tiep trong JSX, nen dung map hoac condition expression.

Phan 3 - Props

Props la du lieu component cha truyen xuong component con.

function ProductCard({ product }) {
  return (
    <article className="card">
      <img src={product.image} alt={product.name} />
      <h3>{product.name}</h3>
      <p>{product.price.toLocaleString("vi-VN")} d</p>
    </article>
  );
}
Nho: Props la dau vao. Component con khong sua truc tiep props.

Phan 4 - Event

Event giup UI phan ung khi nguoi dung thao tac.

function ProductCard({ product }) {
  const onAddToCart = () => {
    console.log("Them vao gio:", product.name);
  };

  return (
    <button onClick={onAddToCart}>Them gio</button>
  );
}
  • Truyen ham vao onClick, khong goi ham ngay khi render.
  • Moi card co event rieng theo product cua card do.

Phan 5 - Map list + key

Dung map de render danh sach thay vi copy-paste 10 card.

function ProductList({ products }) {
  return (
    <section className="grid">
      {products.map((p) => (
        <ProductCard key={p.id} product={p} />
      ))}
    </section>
  );
}
key nen la id duy nhat va on dinh. Khong uu tien dung index.

Bai tap buoc 2 - ProductCard + ProductList

// ProductCard.jsx
export function ProductCard({ product }) {
  return (
    <article className="card">
      <img src={product.image} alt={product.name} />
      <h3>{product.name}</h3>
      <p>{product.price.toLocaleString("vi-VN")} d</p>
      <button onClick={() => console.log(product.id)}>Xem nhanh</button>
    </article>
  );
}

// ProductList.jsx
export function ProductList({ products }) {
  return products.map((item) => (
    <ProductCard key={item.id} product={item} />
  ));
}

Checklist dat cuoi buoi

1) Tao duoc ProductCard tach rieng file/component.
2) ProductCard nhan props va hien dung ten-gia-anh.
3) Co event onClick tren nut.
4) Render bang map, khong copy 10 card bang tay.
5) Hien dung 10 item mock data.
6) Moi item co key hop le.
7) Nhi demo duoc trong 3 phut.
8) Tra loi duoc props la gi, map de lam gi.

Loi thuong gap + cach sua nhanh

Khong hien list

Thuong do quen return trong map.

Canh bao key

Them key bang id thay vi index.

Event khong chay

Check onClick dang truyen ham dung cach chua.

Anh khong hien

Check src va alt, test bang mot URL chac chan ton tai.
